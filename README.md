# Patria Grande - Juego de Memoria

¡Bienvenido al juego de memoria de la Patria Grande! 🌎

## 🎮 Cómo jugar

Este es un divertido juego de memoria con temática latinoamericana donde debes encontrar los pares de banderas de países hermanos.

### Instrucciones:
- 🌟 Haz clic en las cartas para voltearlas
- 🌟 Encuentra las banderas que forman pareja
- 🌟 Solo puedes voltear 2 cartas a la vez
- 🌟 Memoriza las posiciones para hacer menos movimientos
- 🌟 ¡Encuentra los 8 pares para ganar!

### Países incluidos:
🇦🇷 Argentina | 🇧🇷 Brasil | 🇲🇽 México | 🇨🇴 Colombia | 🇵🇪 Perú | 🇨🇱 Chile | 🇺🇾 Uruguay | 🇵🇾 Paraguay

## 🚀 Cómo ejecutar el juego

1. Abre el archivo `index.html` en tu navegador web
2. ¡Comienza a jugar inmediatamente!

O si prefieres usar un servidor web local:
```bash
python3 -m http.server 8000
```
Luego visita `http://localhost:8000` en tu navegador.
